# 🌤️ Weather CLI Tool

A Python command-line app that fetches real-time weather data.

## Features
- Real-time weather using OpenWeatherMap API
- Handles multiple city searches in one session
- Error handling for invalid cities and network issues
- Logs all searches to history.txt

## How to Run
pip install requests
python weather.py

## Technologies
- Python
- REST API
- File I/O
